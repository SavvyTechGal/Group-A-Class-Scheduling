The purpose of this project is to provide registrar with an efficient method of scheduling classes, for the CIS and CSC departments. The user will input the number of sections for each course to be scheduled by the algorithm. The algorithm 
will assign a room number and time for the course, based on the available time slots for the rooms. The program will display a table to the user of the sections that have been scheduled, with 
room number, day of the week, timeslot, course number, and section number.The user will also have the ability to view a weekly/daily views of each room independently, as well as by course number.
